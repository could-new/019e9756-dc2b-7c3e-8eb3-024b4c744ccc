# Ghant Punjabi Songs

ਇਹ ਇੱਕ ਸ਼ਾਨਦਾਰ ਪੰਜਾਬੀ ਗੀਤਾਂ ਵਾਲੀ ਐਪ ਹੈ। ਇਸ ਐਪ ਵਿੱਚ ਤੁਸੀਂ ਆਪਣੇ ਮਨਪਸੰਦ ਪੰਜਾਬੀ ਗੀਤਾਂ ਦਾ ਆਨੰਦ ਮਾਣ ਸਕਦੇ ਹੋ।

## ਫੀਚਰ
- ਪੰਜਾਬੀ ਗੀਤਾਂ ਦੀ ਸੂਚੀ
- ਗੀਤ ਚਲਾਉਣ ਵਾਲਾ ਇੰਟਰਫੇਸ
- ਮਨਪਸੰਦ ਗੀਤਾਂ ਨੂੰ ਸੇਵ ਕਰਨਾ (ਆਉਣ ਵਾਲੇ ਸਮੇਂ ਵਿੱਚ)

## ਕਿਵੇਂ ਚਲਾਈਏ
1. ਇਸ ਪ੍ਰੋਜੈਕਟ ਨੂੰ ਡਾਊਨਲੋਡ ਕਰੋ।
2. `flutter run` ਕਮਾਂਡ ਵਰਤ ਕੇ ਐਪ ਚਲਾਓ।

## CouldAI ਬਾਰੇ
ਇਹ ਐਪ [CouldAI](https://could.ai) ਦੁਆਰਾ ਤਿਆਰ ਕੀਤੀ ਗਈ ਹੈ। CouldAI ਇੱਕ AI ਐਪ ਬਿਲਡਰ ਹੈ ਜੋ ਤੁਹਾਡੇ ਵਿਚਾਰਾਂ ਨੂੰ ਅਸਲ iOS, Android, Web ਅਤੇ Desktop ਐਪਾਂ ਵਿੱਚ ਬਦਲਦਾ ਹੈ। ਇਸ ਵਿੱਚ AI ਏਜੰਟ ਹਨ ਜੋ ਐਪ ਦਾ ਡਿਜ਼ਾਈਨ, ਕੋਡਿੰਗ, ਟੈਸਟਿੰਗ ਅਤੇ ਡਿਪਲਾਇਮੈਂਟ ਕਰਦੇ ਹਨ।
